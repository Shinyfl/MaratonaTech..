Sobre mim

Ola, meu nome é Flávio, tenho 14 anos e meu sonho é ser programador front end, (mas pretendo estudar back end tambem) mas como vou fazer para conquistar esse sonho?, bom, principalmente estudar muito programação mas tambem por conta de uma professora, o nome dela é professora Edina. Descobri a Maratona Tech, e por isso criei esse site para contar mais sobre mim. esse sonho começou, quando vi um cara no tiktok criando um site, isso me deixou fascinado por aqueles codigos assim eu comecei a estudar sobre html, css, e pretendo estudar sobre java script (js). Mas o que é html, css e java script?, simples, são meios de marcação na programação, ela nao chega a ser uma linguagem mas do mesmo jeito é muito divertido programar com elas.

Maratona Tech

Bom, sobre a Maratona Tech, eu fiquei sabendo pela minha professora Edina, ela da aula de teoria geral da administração (tga), ela falou na sala sobre a Maratona Tech e eu resolvi participar. No começo eu comecei fazendo sites mais basico, usando apenas html, nao tinha ciencia nenhuma sobre css, e de um tempo pra ca eu vi que meu site ficava sem cor nenhuma e bem sem graça, assim eu resolvi me aprofundar em css e html, assim eu criei um site com esses dois "codigos", digamos assim. E assim eu vi que o meu site ficou bem mais legal, eu nao tentei hospeda-lo nem nada porque fiz so por diversao, mas da ultima vez que tentei vi que é bem dificil, ainda mais tentar hospedar por um meio gratuito, porque normalmente a hospedagem de sites é paga. tentei pelo git hub e dava um erro generico, tentei por outro site e dava o mesmo erro, assim eu me cansei e nao tensei hospeda-los novamente.

Programação

Programação Back-end é um outro meio que eu acho muito legal, ate mesmo com linguagem para bancos de dados, como py (python), php e dentre outras. eu normalmente entro na minha virtual box (maquina virtual com outro sistema operacional instalado) para brincar, ex: eu pego um iplogger disfarço ele em um link normal (youtube.com ou outro), e mando para os meus amigos, e dou ddos neles (ddos é um meio de congestionar uma rede enviando muitos pacotes ao mesmo tempo), claro isso é uma brincadeira bem besta, mas eu e eles damos muitas risadas fazendo isso. Normalmente a maquina virtual que eu uso é o kali linux, é um sistema operacional usado para cyber segurança, mas bastante usado por hackers tambem.
